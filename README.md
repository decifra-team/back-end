# Back-end Decifra Team

Back-end voltado para a utilização do front da equipe/projeto decifra da disciplina de Gerência de Projetos

# Endpoints
## /login

Endpoint voltado para retorno de um token JWT com expiração de 1 hora para autenticação dos usuários

## /asks

Endpoint que retorna todas as perguntas primordiais para o funcionamento da aplicação

## /answear

Endpoint que recebe o id da pergunta e a nota que o usuário avaliou e armazena isso em uma planilha (temporariamente)
